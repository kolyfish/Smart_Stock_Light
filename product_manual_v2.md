# 🟢 Stock-to-Home Alert 產品手冊 (v1.2)

這是一款專為投資者設計的「硬體實體警報」系統。當您的目標股票達到價格時，家中的 Tapo 智慧燈泡會立即變綠並發出語音提醒，確保您不會錯過任何關鍵交易時刻。

## 🚀 核心功能
*   **即時股價動態**：同步台股（TAIEX）與個股即時走勢。
*   **智慧監控模式**：自動判定「漲破」或「跌破」警報，防止誤鳴。
*   **實體燈光回饋**：
    *   🟡 **黃燈 (常態)**：系統運行中，正在監控股價。
    *   🟢 **綠燈 (觸發)**：股價已達標，提醒入場。
    *   🔴 **紅燈 (警報前緣)**：大盤異常激盪（未來擴充）。
*   **行動端 Web UI**：手機掃描電腦螢幕 QR Code 即可遠端設定。
*   **語音播報 (TTS)**：完整報讀股票代號、中文名稱及目前價格。
*   **系統日誌同步**：首創網頁端 Terminal Log，隨時掌握底層運作狀態。

## 📱 操作指南
1.  **啟動系統**：執行 `./setup_and_run.sh`，電腦會跳出 QR Code。
2.  **手機設定**：
    *   掃描 QR Code 開啟網頁介面。
    *   **股票代號**：輸入 `2330.TW` 或 `鴻海`。
    *   **自定義名稱**：手動輸入中文（例：台積電），確保語音與介面美觀。
    *   **目標價格**：設定您要買入或賣出的價格。
3.  **點擊「更新智慧監控」**：系統會自動抓取現價並判斷警報方向。
4.  **功能演示**：點擊「全功能演示」可測試燈光漸變與語音功能。
