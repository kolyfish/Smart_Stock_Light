# MarketTradeAlertLight (股市亮燈提醒系統) 🟢

這是一個專為非工程師設計的股市監控系統。當您關注的股票跌破目標價時，系統會自動透過 **Tapo 智慧插座** 點亮家中綠燈，並發出語音提醒。

## ✨ 特色
- **一鍵啟動**：專為 Mac 使用者設計的簡易腳本。
- **手機操控**：掃描電腦螢幕上的 QR Code，即可直接用手機設定股票代號與目標價。
- **高品質界面**：全新升級的深色模式 Web UI，高端大氣且直覺。
- **語音警示**：跌破價格時會自動朗讀警報。

## 📖 使用教學與案例

### 🌍 支援多市場監控
系統會自動判斷您輸入的代號，調整更新頻率：
*   **台股** (如 `2330.TW`)：每 10 秒更新一次。
*   **美股** (如 `NVDA`)：每 10 秒更新一次。
*   **虛擬貨幣** (如 `BTC-USD`)：**每秒即時更新**，不錯過任何波動。

### 🟢 與 🔴 的意義
這套系統就像您的 **「全天候交易助理」**，透過燈號直覺告知行情：

*   **🟢 目標獲利價 (Target)**：
    *   **意義**：這是您**「想要出場獲利」**的價格。
    *   **效果**：當價格碰到此目標，燈光轉為 **綠色** 🟢 並語音通知，提醒您「賺錢了，趕快查看」。
*   **🔴 停損警示價 (Stop Loss)**：
    *   **意義**：這是您**「能夠承受虧損」**的底線。
    *   **效果**：當價格跌破此底線，燈光轉為 **紅色** 🔴 並語音警告，提醒您「風險過高，請注意」。
*   **🟣 閃崩警報 (Flash Crash)**：
    *   **定義**：當股價在 **1 分鐘內** 急速下跌超過 **2%** (虛擬貨幣為 2.5%)。
    *   **效果**：這是極度危險訊號，燈光轉為 **紫色** 🟣 並強制語音警告，無論您設定多少價格都會觸發。
*   **🟡 監控中 (Standby)**：
    *   **意義**：價格介於兩者之間，行情波動在預期內。
    *   **效果**：燈光保持 **黃色** 🟡 (或睡眠模式的暗光)，代表「一切正常」。

### ⏳ 5 分鐘冷卻機制 (Cooldown)
*   **為什麼要冷卻？**
    *   當價格在目標價附近上下震盪時，避免警報每秒鐘瘋狂觸發。
*   **如何運作？**
    *   警報觸發一次後，5 分鐘內即使再次達標也不會重複響鈴。
    *   **想立刻再監控？** 手動按下 **「更新智慧監控」** 按鈕，冷卻時間會立即歸零。

### 💡 實戰範例
假設您買入 **BTC (比特幣)** 在 **87,000** 美元：

1.  **設定目標 (🟢)**：希望漲到 **88,000** 賣掉賺錢 👉 輸入 `88000`。
2.  **設定停損 (🔴)**：跌破 **86,000** 就要認賠 👉 輸入 `86000`。
3.  **按下「更新智慧監控」**。
4.  **去睡覺/做其他事**：燈泡顯示溫暖的 **黃光** 🟡。
5.  **狀況 A**：半夜大漲到 **88,000** 👉 燈泡變 **綠色** 🟢 + 語音大喊「價格已達標！」，您醒來獲利了結。
6.  **狀況 B**：行情崩跌到 **85,900** 👉 燈泡變 **紅色** 🔴 + 語音警告「跌破停損價！」，您趕快處理。
7.  **🌙 夜間睡眠模式**：睡覺時按下「睡眠模式」，燈光會變暗至 1% 亮度。
    *   **安心睡覺**：微弱黃光不影響睡眠。
    *   **不漏接行情**：一旦觸發 🟢 綠燈或 🔴 紅燈，系統會**強制全亮**並發出聲音，確保您一定會醒來！

## 技術架構

### 三層架構設計

1. **Hardware Layer** (`tapo_controller.py`)
   - 負責與 TP-Link Tapo L530E 溝通
   - 提供 Thread-safe 的同步介面

2. **Logic Layer** (`stock_monitor.py`)
   - 負責抓取股價與判斷邏輯
   - 實作 Debounce 機制（5 分鐘冷卻時間）
   - 整合 TTS 語音播報

3. **Presentation Layer** (`smart_stock_light.py`)
   - 使用 tkinter 建構 GUI
   - 管理背景監控 Thread
   - 提供即時日誌顯示

### 關鍵技術

- **Threading**：監控迴圈在背景 Thread 執行，不阻塞 GUI
- **Async/Await**：Tapo 控制使用 `plugp100` 的 Async API
- **Logging Handler**：自訂 Handler 將日誌輸出到 GUI
- **Config Persistence**：自動儲存/載入使用者設定

## 注意事項

- 確保 Mac 與 Tapo 燈泡連接到同一個 Wi-Fi 網路
- 首次使用時，建議先測試燈泡連接（可在程式中加入測試按鈕）
- 監控迴圈每 60 秒執行一次，避免過度頻繁的 API 請求
- 防抖動機制確保在 5 分鐘內不會重複觸發警報


## 📦 如何打包成懶人執行檔 (Developer Only)

為了方便非技術背景的客戶使用，我們可以將程式打包成單一執行檔 (`.exe` 或 `.App`)，客戶不需安裝 Python 即可使用。

### 1. 建立 Mac 版本 (`.app`/Binary)
在專案根目錄執行：
```bash
python scripts/build_mac.py
```
完成後，檔案會出現在 `dist/MarketTradeAlertLight`。

### 2. 建立 Windows 版本 (`.exe`)
請在 Windows 電腦上執行：
```cmd
scripts\build_windows.bat
```
完成後，檔案會出現在 `dist\MarketTradeAlertLight.exe`。

### 3. 發布給客戶
只需提供 `dist/` 資料夾內的執行檔即可。
*   **Mac**: 右鍵點選檔案 -> 開啟 (第一次需繞過開發者驗證)。
*   **Windows**: 直接雙擊執行。
