# 🎬 短影音拍攝企劃：【深夜閃崩救星！睡覺也能顧盤的自動化神器】

**核心概念**：利用「深夜閃崩」的危機感，展示「聲光雙重警報」如何拯救投資人的睡眠與荷包。
**風格**：真實、緊張 -> 慶幸、專業解說。
**長度**：約 60 秒

---

## 📦 準備工作 (Setup)
1.  **場景**：昏暗的房間，電腦桌前。
2.  **道具**：
    *   Tapo 智慧燈泡 (放在螢幕旁，非常顯眼的位置)。
    *   電腦螢幕 (顯示 K 線圖或你的軟體介面)。
    *   你自己 (穿睡衣/休閒服)。
3.  **預備動作**：
    *   先將燈泡亮度調低或關閉 (模擬睡覺)。
    *   準備好 Web UI 的「測試紅燈 (停損)」與「測試綠燈 (獲利)」按鈕。
