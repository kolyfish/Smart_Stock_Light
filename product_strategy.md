# 🟢 Stock-to-Home Alert: 產品競爭力與市場定位分析

## 1. 解決的核心痛點 (Pain Points)
許多投資者面臨的不是「資訊不足」，而是「資訊過載」與「生活品質受損」：
*   **盯盤焦慮 (Screen Fatigue)**：長期盯著股市線圖會造成眼睛疲勞，且讓人無法專注於本業或家庭生活。
*   **通知淹沒 (Notification Overload)**：手機上的股票 App 通知經常被 Line 或臉書訊息埋沒，導致錯過關鍵買賣點。
*   **情緒化交易 (Emotional Trading)**：看著股價跳動容易產生恐慌或貪婪。實體燈光的「客觀信號」能幫助投資者維持理性，燈沒亮代表還沒到計畫區間。
*   **操作門檻**：雖然智慧家庭可以透過 IFTTT 等工具連動，但設定過程極其繁瑣，且通常需要額外付費訂閱。

## 2. 與現有競爭品牌的不同 (Differentiation)
目前市場上（如小米、Philips Hue、Govee）雖有燈光控制，但與本專案有顯著區隔：
*   **金融垂直整合**：通用品牌需要透過第三方轉接；本系統「專為獲利設計」，內建台灣大盤監控、閃崩偵測（紫色燈）與停損守護（紅色燈）。
*   **本地化低延遲**：數據直接從 Python 程式端推送至燈泡，不經過漫長的雲端轉運流程，反應更及時。
*   **零成本訂閱**：市面上許多金融連動服務需按月收費，本系統為開源設計，只要有 Python 環境與 Tapo 燈泡即可運行。
*   **多維度警報**：除了顏色切換，還整合了語音播報與網頁日誌，提供視覺、聽覺、數位紀錄三位一體的監控。

## 3. 與「買賣股票 App」的差異 (App vs. Device)
這不是要取代您的下單軟體，而是成為下單軟體的前哨站：
*   **實體與虛擬的區隔**：App 是「交易工具」，這套系統是「生活儀器」。它讓您無需看手機就能知道市場變化，將股市資訊無縫嵌入居家空間。
*   **主動提醒 vs. 被動查詢**：App 需要您去「打開」或「滑掉」通知；燈光是「強迫性的視覺佔領」，只要在房間內，您絕對不會漏掉信號。
*   **情緒緩衝帶**：App 的介面充滿誘惑與急促感；實體燈光作為「寧靜科技 (Calm Technology)」，只有在達到您「預設策略」時才出聲，減少不必要的心理壓力。
